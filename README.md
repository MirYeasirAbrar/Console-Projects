# 🖥️ Console-Projects

## 📌 About This Repository

**Console-Projects** is a personal collection of beginner-friendly, text-based applications built for the terminal (CLI).  
Each project is designed to improve your understanding of core programming concepts using languages like **C** and **C++**, with plans to expand into **Python** and other CLI-capable languages.

This repository serves as:
- 🎯 A **hands-on practice ground** for aspiring developers
- 🧠 A **logic-building toolkit** through simple, real-world applications
- 🛠️ A **showcase of practical terminal utilities**, built with clean code and reusable structure

Anyone who is a student, self-learner, or just revisiting the basics, this repository offers an approachable and structured path to learning console application development.

---
## 📂 Repository Structure

```
Console-Projects/
├── 01-Student Management System/
└── 02-A Phonebook Application/
```



Each folder includes:
- 📄 `README.md`: Description and usage
- 🧠 `main.c` / `main.cpp`: Source code
- 🧪 Input/Output samples or demo files (if needed)

---

## 🚀 How to Use

> 🧰 Requirements: C/C++ compiler (e.g., GCC or g++)

```
# Compile
g++ main.cpp -o app

# Run
./app
```
> 📁 Navigate into each project folder and follow its README.md for usage.
