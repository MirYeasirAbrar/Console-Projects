# 📘 Project 01: Student Management System (Console-Based)

## 🧾 About This Project
The **Student Management System** is a simple, yet functional console-based application written in C. It simulates a basic system that allows the user to manage student records including adding, deleting, updating, and searching data by name or roll number.

It is designed to practice:
- Structured data with `struct`
- Arrays for storage and sorting
- User input/output handling
- Command-line user interface design
- Fundamental fileless database simulation

---

## 🎯 Features
- Add student with name, roll, and CGPA
- Prevent duplicate roll numbers
- View all students in alphabetical order by name
- Search student by name or roll
- Update student details (name, roll, CGPA)
- Delete student record
- Count total number of students
- Gracefully handle invalid inputs

---

## 🧱 Technologies Used
- Language: `C`
- Compiler: `GCC / Clang`
- Environment: Terminal / Console (no external libraries)

---

## 🖥️ How to Run

1. **Clone or download** this repository.
2. **Open** the `main.c` file using your preferred C IDE or text editor (e.g., Code::Blocks, VS Code, CLion).
3. **Compile and Run** the file using the built-in compiler or terminal.

> 💡 No external libraries are required — just a C compiler and a terminal or IDE.
